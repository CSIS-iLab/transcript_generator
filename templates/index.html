Please select an mp3 file to upload:

Before submitting please note:

  • Do not click "Submit" more than once. Doing so will cause the program to start over, which will take longer.
  • Processing your file will take a while. The time it takes to process a file is 15-30% of its length. A 10-minute file would take 1.5 to 3 minutes to process.
  • ChatGTP3 has a word limit. To generate your one-paragraph summary, this tool uses ChatGTP3, which has a limit of ~3000 words. To work within this limit, ChatGTP3 is not fed your entire transcript, but a lengthy first-draft summary generated by a separate AI tool.
  • You will need to review the transcript for errors. The AI will generate a complete transcript for you, but it may not spell every name or location correctly (ex: "Dr. Hammery"). If your event contains multiple speakers or presenters, it will label different speakers "Speaker A", "Speaker B", and so on. While it does a good job of differentiating between speakers, it does not "know" who is speaking and cannot label them by name.
  • You can still try ChatGTP4. ChatGTP4 is able to process up to 25,000 words of text and create more sophisticated summaries and analysis. ChatGTP4 is not yet available for developers to build with, but you may interact with it via ChatGTP+ and Microsoft's new Bing chat. If you would like, you will be able to copy and paste the transcript that is generated for you on the next page into one of those tools.
  • You can ask for a new summary. On the next page, if you are unhappy with your summary you will have the chance to request that the AI try again. You will have the option to use the same default prompt, or submit your own prompt. Note the variation in the result will be limited, in order to prevent the AI from inventing things that are inaccurate.
